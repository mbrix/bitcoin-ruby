    class Bitcoin::Builder::TxInBuilder

    create a Bitcoin::Protocol::TxIn used by Bitcoin::Builder::TxBuilder#input.

    inputs can be either ‘coinbase’, in which case they only need to specify #coinbase, or they have to define a #prev_out, #prev_out_index and signature key.

    create the txin according to values specified via DSL

    # File lib/bitcoin/builder.rb, line 214
def txin
  @txin.prev_out = (@prev_out ? @prev_out.binary_hash : "\x00"*32)
  @txin.prev_out_index = @prev_out_index
  @txin.sequence = @sequence || "\xff\xff\xff\xff"
  @txin
end
